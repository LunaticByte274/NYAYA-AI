# Nyaya AI: Fairness Intelligence System

A competition-winning platform that detects, explains, and mitigates bias in automated decision-making systems and human text using Google Gemini and deterministic statistical analysis (Disparate Impact).

## Quick Start
1. Add your Google Gemini API key to the `.env` file.
2. Run `npm run install:all` from the root directory.
3. Run `npm run dev` to launch the full stack (Next.js UI on port 3000, FastAPI on port 8000).
